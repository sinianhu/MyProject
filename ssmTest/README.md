# liujilu-ssm
Maven + SpringMVC+Spring+MyBatis-Plus+Freemarker 框架实现的一个后台网站

> 框架支持Freemarker 和  JSP 双View展示（优先找Freemarker）。

## 如何使用：
`git clone git@github.com:Jandaes/liujilu-ssm.git`

导入IDEA中、也可直接导入Eclipse工具

## 代码生成
内置Mybatis 自动生成代码、查看代码
